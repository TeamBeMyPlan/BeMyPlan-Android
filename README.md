# BeMyPlan-Android
#### 내 취향에 맞는 기여행자의 여행 일정 컨텐츠 구매 서비스



## 사용한 기술 스택
- MVC +MVVM
- Coroutines
- DataBinding
- Kakao SDK
- KakaoMap SDK



## 폴더링 구조

	BeMyPlan
	 ┣ 📂application
	 ┣ 📂data
	   ┣ 📂api
	   ┣ 📂entity
	   ┣ 📂repository
	 ┣ 📂ui
	   ┣ 📂adapter
	   ┗ 📂viewmodel
	 ┗ 📂util


## 작업 분담

#### 승현

```
여행지, 스크랩, 정렬, 구매전 여행일정, 로그인, 회원가입
```

#### 정훈

```
구매후 여행일정, 지도, 마이플랜
```

#### 용민

```
홈, 결제
```

#### 영민

```
자문 및 멘토
```
